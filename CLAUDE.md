## Build

- `bundle exec jekyll serve` to run locally.
- Deployed via GitHub Pages.

## Conventions

- Default post layout is configured in `_config.yml` — no need for `layout: post` in front matter.
- CLAUDE.md is excluded from the Jekyll build via `_config.yml`.
